# Changelog

All notable changes to this project are documented here.

## [1.0.0] — 2026-05-08

### Added

- Five alien AI agents (Affa, Ponnar, Monka, Orthon, Zemkla) powered by GPT-4o
- RAG pipeline over 3,037 pages of declassified FBI, CIA, DoD, and AARO documents
- PostgreSQL full-text search across 2,416 indexed document chunks
- Source citations on every alien response linking directly to the source PDF page
- Persistent chat history per session per alien
- UAP Incident Globe — interactive 3D map of 40 declassified incidents
- Declassified Media Archive — 23 UAP videos and FLIR thermal imaging stills
- Marquee of verbatim quotes sourced directly from government files
- Mobile-responsive layout across all pages
- OpenAPI-first backend with generated React Query hooks and Zod validation
